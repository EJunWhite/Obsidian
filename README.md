### Folder Rule

```Fold
1.Tasks
2.Policy
3.Meeting
4.Template(only obsidian)
```

| No   | Name     | Desc                  |
| ---- | -------- | --------------------- |
| 1    | Tasks    | 협업 관련 - 일 분배 및 협업 등등  |
| 2    | Policy   | 정책 관련 - 정의, 수립, 과정 등등 |
| 3    | Meeting  | 회의 관련 - 티타임 등등        |
| 9999 | Template | 옵시디언 - 템플릿            |
